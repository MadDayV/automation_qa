# Selenium & Python Automation Framework

Professional test automation suite demonstrating scalable E2E testing architecture for web applications.

## 🛠 Tech Stack
- **Language:** Python 3.x
- **Core Tool:** Selenium WebDriver
- **Test Runner:** Pytest
- **Design Pattern:** Page Object Model (POM)

## 🏗 Framework Architecture
- **Page Object Model:** Logic is separated from test scripts for high maintainability and readability.
- **Base Page Class:** Implemented a base class with common methods to reduce code duplication.
- **Configuration:** Flexible test setup via Pytest fixtures and `conftest.py`.
- **Selectors:** Optimized use of CSS selectors and XPATH for robust element location.

## 🚀 How to Run
1. **Clone the repo:**
   ```bash
   git clone https://github.com
   ```
2. **Install requirements:**
   ```bash
   pip install -r requirements.txt
   ```
3. **Run tests:**
   ```bash
   pytest
   ```
